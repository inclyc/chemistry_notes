    \subsection{根据反应设计电池}

    电池与反应的互译，一般先寻找阳极和阴极，在复核。

    \subsubsection{非氧化还原反应的电极反应设计}

    \begin{reaction*}
        AgCl(s) + I- (a1) -> AgI(s) + Cl- (a2)
    \end{reaction*}

    可以给两边都加一个$\ch{Ag}$

    \begin{reaction*}
        AgCl(s) + I- (a1) + Ag -> AgI(s) + Ag + Cl- (a2)
    \end{reaction*}
